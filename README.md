# 전화대장군 Frontend

2024년 제6회 K-디지털 트레이닝 해커톤 - 팀 음성감독원

## 프로젝트 소개

노년층 보이스피싱 피해 방지를 위한 AI 기반 실시간 통화 분석 서비스

## 주요 기능

- 실시간 통화 감지 및 음성 데이터 스트리밍
- FCM 푸시 알림 수신 (보이스피싱 위험 경고)
- 보호자 연락처 등록 및 관리
- 직관적인 UI/UX (노년층 친화적 디자인)

## 기술 스택

- React Native 0.76
- TypeScript
- React Navigation
- Firebase Cloud Messaging
- React Native Audio Recorder Player
- React Native CallKeep
